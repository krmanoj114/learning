Steps to install docker on windows:-
1>Open - Turn windows feature on or off
2>Select following cgeckbox ( "Hyper-V",  "Virtual Machine Platform" & Windows Subsystem for Lynux)
3>Click "OK" Button
4> Restart System
5> Check the wsl status
# wsl --status (If wls is not installed or updated than follow the following steps mentioned in https://learn.microsoft.com/en-us/windows/wsl/install-manual )
6> Download docker for desktop for windos from https://docs.docker.com/engine/install/
7> install the downloaded file as an admin
8> Opem the docker by clicking on docker icon
9> Register youself on docker, if not registered

Managing network using CLI
--------------------------
#To check the network info
docker info   --provide info about network

#Command to create a docker network of bridge type
docker network create -d bridge network1

#To run Mysql
docker run --name mysql -e MYSQL_ROOT_PASSWORD=cfg@1234 -d mysql/mysql-server 

# To add container to current network
docker network connect mynet1 mysql 

#To inspect the container
docker inspect mysql | grep "IPAddress"
